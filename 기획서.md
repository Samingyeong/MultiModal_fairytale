# 장애아동 맞춤 멀티모달 동화 학습 플랫폼

---

## 아이디어 요약

**"모두를 위한 동화 — 듣고, 보고, 쓰고, 손으로 말하는 멀티모달 동화 학습 플랫폼"**

청각장애·언어발달 지연 아동을 포함한 모든 아이들이 동화를 통해 한국어를 배울 수 있도록, 영상·자막·수어·필기 인식을 하나의 플랫폼에 통합한 AI 기반 학습 서비스입니다.

---

## 기획 배경

### 문제 정의

국내 장애아동(청각장애, 언어발달 지연 등)은 일반 아동 대비 언어 학습 기회가 현저히 부족합니다. 기존 동화 콘텐츠는 음성 중심으로 설계되어 있어 청각장애 아동이 접근하기 어렵고, 수어 병행 콘텐츠는 극히 드뭅니다. 또한 다문화 가정 아동이나 외국어권 아동도 한국어 동화 학습에 높은 진입 장벽을 경험합니다.

### 개발 동기

- 국립어린이청소년도서관(NLCY)이 제공하는 **다국어 동화 영상**(한국어·영어·베트남어·중국어·태국어·몽골어)이 공공데이터로 개방되어 있으나, 이를 학습에 활용하는 서비스가 없음
- NIA(한국지능정보사회진흥원)의 **한국 수어 키포인트 라벨링 데이터셋**이 공개되어 있으나 실제 서비스에 연결된 사례가 없음
- 아이들이 단순히 동화를 '보는' 것을 넘어, 단어를 클릭하고 사전을 찾고 직접 써보는 **능동적 학습 경험**이 필요함

### 해결 방향

공공 동화 영상 + AI 형태소 분석 + 수어 키포인트 데이터 + 필기 인식을 결합하여, 장애 유무와 언어 배경에 관계없이 누구나 동화로 한국어를 배울 수 있는 플랫폼을 구축합니다.

## 아이디어 주요 내용

### 핵심 구조

```
[공공 동화 영상 (NLCY)]
        ↓
[다국어 자막 (VTT) 파싱]
        ↓
[AI 형태소 분석 (KoNLPy + Groq LLM)]
        ↓
[단어 클릭 → 사전 조회 (한국어기초사전 API)]
        ↓
[수어 키포인트 매핑 (NIA 데이터셋)]
        ↓
[필기 따라쓰기 (Canvas + Google Vision API)]
```

### 주요 기능

**1. 멀티모달 동화 뷰어**
- 국립어린이청소년도서관 동화 영상을 영상+자막 병렬 화면으로 제공
- 한국어·영어·베트남어·중국어·태국어·몽골어 6개 언어 자막 실시간 전환
- 영상 재생 위치와 자막이 자동 동기화

**2. 단어 학습 (클릭 사전)**
- 자막의 모든 단어를 클릭하면 AI 형태소 분석으로 기본형 추출
- 한국어기초사전 API 연동으로 뜻·품사·난이도 등급 즉시 표시
- "알아요 / 몰라요" 버튼으로 개인 단어장 자동 구성
- 자막 로드 시 전체 문장 일괄 분석(batch) → 클릭 즉시 응답

**3. 수어 연동 (개발 중)**
- NIA 한국 수어 키포인트 데이터셋(2D/3D 관절 좌표) 활용
- 동화 자막의 핵심 단어를 수어 동작으로 시각화
- 청각장애 아동이 수어로 동화 내용을 이해할 수 있도록 지원

**4. 필기 따라쓰기**
- 단어 클릭 시 캔버스에 회색 가이드 글자 표시
- 아이가 위에 직접 손가락/펜으로 따라 쓰기
- Google Vision API 연동으로 필기 인식 및 정확도 피드백

**5. 카테고리별 동화 탐색**
- 한국전래동화 / 외국전래동화 / 창작동화 / K-그림책 분류
- 문화공공데이터광장 API 자동 동기화 (매일 새벽 3시 cron)
- 제목·작가 검색, 페이지네이션

### 실행 전략

| 단계 | 내용 |
|------|------|
| 1단계 (현재) | 동화 뷰어 + 단어 학습 + 필기 따라쓰기 프로토타입 완성 |
| 2단계 | 수어 키포인트 → 3D 아바타 애니메이션 연동 |
| 3단계 | 학부모 대시보드 (학습 이력, 오답 단어 관리) |
| 4단계 | 장애아동 특수교육 기관 파일럿 적용 및 피드백 반영 |

## 프로젝트 계획

### 활용 데이터

| 데이터 | 출처 | 활용 방식 |
|--------|------|-----------|
| 다국어 동화 영상·자막 | 국립어린이청소년도서관 (NLCY) | 동화 콘텐츠 및 VTT 자막 제공 |
| 동화 메타데이터 | 문화공공데이터광장 API (API_LIB_048) | 책 목록 자동 동기화 |
| 한국 수어 키포인트 | NIA 한국지능정보사회진흥원 | 수어 동작 시각화 |
| 한국어 사전 | 국립국어원 한국어기초사전 API | 단어 뜻·품사·등급 조회 |

### 활용 AI 기술

| 기술 | 도구 | 역할 |
|------|------|------|
| 한국어 형태소 분석 | KoNLPy (Okt) | 자막 단어 기본형 추출, 품사 태깅 |
| LLM 형태소 보정 | Groq (Llama 3.1) / Gemini 2.0 Flash | 문맥 기반 동형이의어 판별, 수어 핵심 단어 추출 |
| 필기 인식 | Google Vision API | 아이 필기 텍스트 인식 및 정확도 평가 |
| 수어 키포인트 | NIA 라벨링 데이터 (2D/3D 관절 25+21점) | 수어 동작 재현 |

### 기술 스택

- **프론트엔드**: React 19 + TypeScript + Vite
- **백엔드**: Node.js + Express + SQLite (better-sqlite3)
- **AI/NLP**: Python (KoNLPy), Groq API, Gemini API
- **인프라**: 로컬 이미지 캐시, 자동 동기화 cron

### 현재 구현 수준 (프로토타입)

- ✅ 동화 목록 자동 수집 및 DB 저장
- ✅ 다국어 동화 영상 + 자막 뷰어
- ✅ 단어 클릭 → 형태소 분석 → 사전 조회
- ✅ 개인 단어장 (알아요/몰라요)
- ✅ 필기 따라쓰기 캔버스
- ✅ 홈화면 추천 동화 (API 연동)
- 🔄 수어 키포인트 → 아바타 애니메이션 (개발 중)
- 🔄 학부모 대시보드 (기획 중)

### 본선까지 기획 가능한 수준

- NIA 수어 키포인트 데이터를 Three.js 또는 MediaPipe 기반 3D 아바타에 매핑하여 동화 자막 단어에 수어 애니메이션 표시
- 아이의 학습 이력 분석 → 취약 단어 자동 복습 추천
- 프로토타입 시연 가능 (현재 로컬 환경에서 전체 기능 동작 확인)

## 기대효과

### 사회적 효과

- **교육 접근성 향상**: 청각장애·언어발달 지연 아동이 수어와 시각 자막을 통해 동화 학습에 동등하게 참여 가능
- **다문화 가정 지원**: 6개 언어 자막 제공으로 외국어권 아동의 한국어 습득 경로 확대
- **공공데이터 활용 극대화**: 국립도서관·NIA·국립국어원 등 분산된 공공 데이터를 하나의 학습 경험으로 통합

### 정책적 시사점

- 특수교육 디지털 전환 정책과 연계 가능 (장애아동 맞춤형 AI 보조 학습 도구)
- 국립어린이청소년도서관 다국어 동화 콘텐츠의 실질적 활용 모델 제시
- NIA 수어 데이터셋의 서비스 적용 가능성 실증

### 시민 인식 변화

- 장애아동 학습 도구가 '특수 기기'가 아닌 일반 스마트폰·태블릿에서 동작하는 서비스로 인식 전환
- 비장애 아동도 함께 사용하는 통합 플랫폼으로, 장애에 대한 자연스러운 이해와 공감 형성
- 부모와 아이가 함께 단어를 배우고 수어를 익히는 가족 학습 문화 조성
